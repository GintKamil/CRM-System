Сайт CRM-System

В проекте используется паттерны Modular Monolith, MVC
База данных - PostgreSQL

Два метода запуска - Web версия и Api версия
Разработана авторизация и регистрация пользователя, crud операции для работы с клиентами и задач, отчеты для просмотра информации, админ.панель

Существуют несколько ролей для пользователей, в зависимости от роли функционал изменяется
