Сайт CRM-System

## Для чего
CRM система разработана для управления клиентами и задачами, так же есть отчетность по задачам.

## 🛠 Технический стек
- Frontend: HTML, CSS, JS
- Backend: C#, ASP.NET Core
- Database: PostgreSQL
- Auth: Cookie (Web версия) + JWT (Api версия)

В проекте используется подход Modular Monolith с использованием шаблона MVC и принципов SOLID (часть)
База данных - PostgreSQL

## Возможности

Два метода запуска - Web версия и Api версия
Разработана авторизация и регистрация пользователя, crud операции для работы с клиентами и задач, отчеты для просмотра информации, админ.панель

Существуют несколько ролей для пользователей, в зависимости от роли функционал изменяется

Добавлены уведомления пользователей через SignalR
Добавлены комментарии пользователей в задачах
